# CS52 Bootcamp - Day 1

## Setup

```bash
git clone https://github.com/Alezander9/CS52W1D1.git
cd CS52W1D1
npm install
npm run dev
```

Open the link shown in your terminal (usually http://localhost:5173).

## Your Page

Find your file in `src/pages/<your-name>`.

Edit that file, save, and your changes appear instantly in the browser at `/<your-name>`.
